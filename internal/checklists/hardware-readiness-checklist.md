## Checklist

### Procurement

- [ ] Hardware list confirmed from Kickoff (printers, scanners, workstations, mobile devices)
- [ ] Devices quoted / ordered (or customer-supplied confirmed)
- [ ] Delivery / inventory received for go-live cohort

### Install and smoke test

- [ ] Drivers / pairing completed on go-live workstations
- [ ] Mobile printer print test (citations / forms as in scope)
- [ ] Barcode / evidence scanner test (if in scope)
- [ ] Label / receipt printer test (if in scope)
- [ ] Spare consumables (paper, labels, toner) available or ordered

### Sign-off

- [ ] No hardware blockers for Go Live (or deferred with risk acceptance)
