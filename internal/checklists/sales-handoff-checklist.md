## Checklist

### Commercial

- [ ] Contract / SaaS path clear (incl. CJIS path if required)
- [ ] Modules / products in scope listed
- [ ] Migration fee / pricing commitments noted ([Migration Pricing Policy](../policies/migration-pricing.md))
- [ ] Target go-live window (if known)

### Scope flags

- [ ] Historical data migration: Y / N / Unknown
- [ ] Integrations expected (list or “none”)
- [ ] Hardware expected (list or “none”)
- [ ] Migration assessment status (not started / in progress / approved / N/A)

### Contacts

- [ ] Primary customer contact
- [ ] Technical / IT contact (if known)
- [ ] Sales owner for questions
- [ ] Implementation owner assigned

### Risks

- [ ] Known risks / constraints listed
- [ ] Kickoff date proposed or set
