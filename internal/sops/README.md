# Standard operating procedures (SOPs)

Repeatable procedures organized by [Customer Value Engine](../customer-value-engine/README.md) phase.

## SOP structure

Every SOP should use this structure:

1. Purpose
2. Scope
3. Owner
4. Trigger
5. Preconditions
6. Inputs
7. Outputs
8. Tools
9. Procedure (numbered steps)
10. Verification
11. Failure and escalation
12. KPIs
13. Related documents
14. Continuous improvement (Simplify → Scale + product opportunities)
15. Change history
