## Purpose

Transfer a signed customer from Sales to Implementation with enough commercial and scope clarity to start kickoff without re-discovering the deal.

---

## Inputs

| Input | Source |
|-------|--------|
| Signed SaaS (+ CJIS path as required) | Acquire / Contract |
| Scope summary (modules, migration Y/N) | Proposal / contract |
| Contacts (customer + Sales) | CRM / deal notes |
| Migration assessment status (if any) | [Legacy System Migration Assessment](../../assessments/legacy-system-migration-assessment.md) |
| Pricing / fee commitments | [Migration Pricing Policy](../../policies/migration-pricing.md) |

---

## Activities

1. Package handoff packet (scope, dates, contacts, risks).  
2. Confirm Implementation owner and capacity.  
3. Flag migration / integration / go-live constraints.  
4. Schedule kickoff.
