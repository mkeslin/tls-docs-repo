## Purpose

Run every engagement with one repeatable lifecycle: handoff through hypercare. Each phase has a measurable outcome. Standards and SOPs support the phases—they are not the navigation.

## Principles

1. Organize Deliver around the **lifecycle**, not tool names.  
2. Bootstrap **stops** at Infrastructure; agency setup is Configuration.  
3. Data migration includes assessment, packages, validation, and acceptance—not only running scripts.  
4. Standards define done; SOPs define how; checklists prove it.
