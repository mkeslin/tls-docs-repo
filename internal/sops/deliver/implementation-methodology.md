## Purpose

The end-to-end method for taking a signed customer from handoff through hypercare. Migration, bootstrap, and configuration are **phases inside** this methodology—not the whole of Deliver.

```text
Sales Handoff
  → Kickoff
  → Infrastructure (bootstrap)
  → Data Migration (if in scope)
  → Configuration
  → Integrations
  → Training
  → Go Live
  → Hypercare
```

## Principles (starter)

1. Bootstrap before configuration; validate migration before acceptance.  
2. Standards define “done”; SOPs define “how.”  
3. Customer-specific work is configuration—not one-off infra.
