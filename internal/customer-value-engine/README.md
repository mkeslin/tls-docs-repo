# Customer Value Engine

How Thin Line creates and delivers customer value — from first awareness through advocacy.

The Customer Value Engine is the operating model for the business: five phases, nineteen stages, each with owners, tooling, capacity, maturity, and improvement ideas.

| Page | Purpose |
|------|---------|
| [Overview](overview.md) | Model summary, maturity scale, and improvement framework |
| [Customer value stream](customer-value-stream.md) | Phase map and stage index |

## Phases

| Phase | Outcome |
|-------|---------|
| [Acquire](acquire/README.md) | Signed customer ready to implement |
| [Deliver](deliver/README.md) | Customer live on Thin Line Platform |
| [Operate](operate/README.md) | Reliable support and product updates |
| [Expand](expand/README.md) | Deeper adoption and additional modules/sites |
| [Advocate](advocate/README.md) | References, referrals, and community |

Procedures for these stages live under [SOPs](../sops/README.md).
