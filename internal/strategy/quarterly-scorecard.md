# Quarterly scorecard

## Purpose

Track a small set of leading and lagging indicators that show whether Thin Line is building the platform, winning and delivering customers, and supporting them well.

## How to use

1. Update at the end of each quarter (or monthly if preferred — **Decision needed:** cadence).
2. Keep metrics few and actionable.
3. Link each metric to a value-stream stage where possible.
