# Plans

Implementation plans for backlog items and named epics. Plans are **versioned text** in this docs repo, not chat-only.

## Naming

- **Pattern:** `BL-###-kebab-case-short-title.md` when the item has a backlog id (preferred).
- **Examples:** `BL-015-accounting-grid-prints.md`, `BL-001-oca-february-verify.md`
- If an item has no id yet, `kebab-case-short-title.md` is fine until [prioritized.md](../prioritized.md) assigns **`BL-###`**.

Avoid spaces in filenames so links work everywhere.

## Optional frontmatter

```yaml
---
backlog: "BL-### · Module · Work item name"
status: draft   # draft | ready | in-progress | done | superseded
created: YYYY-MM-DD
---
```
