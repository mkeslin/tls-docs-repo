## General Changes

- Added a new Thin Line Software landing page
- Added user feedback by SMS
- Added "Other Conditions" field to citation and mobile citation
- Added Avery 8160 label support to applicable court documents
- Added the ability to print incident reports without call information
- Added "Create Call from Citation" workflow (initial)
- Added a citation-validation bypass for closing citations that fail validation (workflow status 10)
- Added themes framework with theme selection (currently disabled while iterated on)
- Updated CAD, mobile citation, navigation, and toolbar styling
- Updated narrative editing so only officers can add or edit narratives, with a new claim for narrative administrators
- Restricted modification of incident narratives, including locking after approval
- Reorganized the person-involved report
- Updated rich-text font color rendering and the read-only narrative behavior
- Reorganized incident narrative tab; primary first, then by creation date
- Updated incident detail report with new layout
- Added an option to show or hide age in records (with reference date for accuracy)
- Added the ability to copy a user's email or phone to the clipboard
- Updated audit and security logs to display seconds and milliseconds
- Allowed multiple audit timeline entries to be expanded at once
- Added the ability to remove a citation that is stuck in sync
- Made other small improvements and fixes

## Court Violations

- Added a comprehensive set of court documents
- Added court violation workflow status, validations, and search updates
- Added duplicate court violation prevention
- Added flag-for-review fields on offense court violations
- Added offense-to-court-violation mappings, fees, and fine ranges
- Added offense court violation search, models, and management dialogs

## Accounting

- Added accounting fees and schedules with full add/edit, search, and association to offenses
- Added accounting accounts, transactions, transaction sets, balances, and group-by support
- Added the ability to apply payments
- Added manual fees dialog
- Stubbed in payment plans
- Added accounting icon and accounting transaction search

## OCA & Reports

- Added OCA report (XML generator and UI)
- Added State Quarterly Report
- Added DPS Conviction Report
- Added new Incident Detail Report

## Mobile Citation

- Disallowed UNKNOWN race, ethnicity, and gender on mobile citations
- Fixed initial appearance date not populating
- Fixed initial appearance date on mobile citation print

## NIBRS

- Added NIBRS 2025.0 export with new models and validations
- Updated criminal activity and gang information handling
- Added unfounded incident logic
- Updated and fixed several NIBRS validations

## CAD

- Added new call unit statuses with toggling and mobile CAD support
- Added a CAD clock
- Fixed unit status not updating until screen refresh
- Fixed call unit times when edited
- Fixed click propagation between "All Calls" and "My Calls"
- Fixed self-dispatch sorting
- Filtered required-narrative task records by the agency of the unit

## Evidence & Property

- Added evidence label print logic and layout
- Fixed deletion of property associated with an evidence record

## Architecture Changes

- Switched API URLs to kebab-case
- Re-added Swagger / OpenAPI documentation
- Removed legacy reporting Web API
- Updated authentication logic and attachment endpoint security
- Renamed CJIS to EDR; stubbed in EDR export and validation
- Stubbed in Vue Flow
- Reorganized server solution and pipeline files
- Added new offense codes (UCR mappings, Sept. 2025 offenses)
- Updated client-side libraries
- Updated server-side libraries
- Made other small improvements and fixes
