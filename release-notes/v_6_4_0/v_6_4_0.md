# Version 6.4.1

## Jail Management (JLM)

- Improved observation pass scheduling for automated and manual cell checks
- Added ability to cancel an observation pass, with schedules adjusted correctly
- Historical cell-check views now include released inmates
- Added quick-select buttons for common cell-check observations
- Improved cell-check notes, row selection, and keyboard shortcuts in the Pass Runner
- Added notification sounds and in-app audio settings for jail tasks
- Added real-time "booking in progress" notifications on the JLM board
- Improved task alert sounds in the jail work queue

## Jail Intake

- Improved offense code and level display on booking charges and reports
- Improved arresting officer name display on booking reports
- Improved date and time handling on jail intake reports
- Improved booking charge formatting on reports

## Incidents

- Added a Juvenile incident tag
- Made incident tags editable (except Cargo Theft)
- Improved how incident tags display on incidents
- Locked selected incident tags used for reporting control

## Accounting & Collections

- Improved date range selection on collections and bond reports and batch screens
- Standardized accounting fee descriptions
- Added an Animal Control fee

## Administration & Mobile Citation

- Added a mobile citation print preview on agency admin
- Improved help text and layout on agency admin general settings

## General Changes

- Made other small improvements and fixes
