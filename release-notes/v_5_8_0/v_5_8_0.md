## General Changes

- Updated court logic
- Stubbed in show cause letter
- Added the ability to remove a citation that is stuck in sync
- Updated initial appearance date logic to ignore citations with only warnings
- Fixed deletion of property attached to an evidence record
- Fixed an IBRS validation error
- Fixed lost-and-found script
- Made other small improvements and fixes

## CAD

- Filtered required-narrative task records by the agency of the unit
- Fixed unit status not updating in CAD until screen refresh
- Fixed an issue where, when an officer was assigned to multiple units, the wrong unit could appear active

## Architecture Changes

- Updated client-side libraries
- Updated server-side libraries
- Made other small improvements and fixes
