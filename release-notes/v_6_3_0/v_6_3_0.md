## General Changes

- Added session security improvements for CJIS IA-11 compliance
- Added a security banner for active sessions
- Made other small improvements and fixes

## Jail Intake

### Reports

- Added a Booking Medical Report (PDF) with medical history, allergies, and medications
- Added a Booking Mental Health Report (PDF) from mental health screening
- Added a Booking Classification Notice Report (PDF), including the classification flowchart
- Added a Booking Custody Report and a Booking Property Report with consistent layouts
- Added a Booking Detail Report
- Added PDF generation for jail intake step reports, with validation before generation
- Improved person details and charge line presentation on reports
- Improved layout consistency, headers, footers, and section titles across jail intake reports

### Functionality

- Added ability to delete a booking from the booking details view
- Improved release booking security and permissions
- Added validation enhancements to the custody step
- Improved name handling on jail intake entry and booking steps
- Reorganized jail intake search options and results for clarity
- Renamed "Medical Log" to "Medication Log" across the application
- Added PDF view and download in the Medication Log and Observation Pass searches

### UI / Layout

- Improved layout and styling on the custody, medical, mental health, and classification steps
- Added a read-only final view on the classification step after completion
- Removed the housing assignment step pending further work
- Disabled the DOCS step pending further work

## Jail Management (JLM)

- Added an Offsite Transfer Wizard with checklist and ability to abandon a draft transfer
- Added a Release Booking dialog with identity confirmation
- Added roster PDF export from the JLM board
- Improved observation pass scheduling, including round cadence and missed-round handling
- Medications and related tasks are deactivated when a custody episode is released
- Simplified offsite transfer workflow (inter-agency transfer no longer required)
- Improved board loading, polling, and drag-and-drop responsiveness
- Streamlined navigation from JLM cards

## Court Violations

- Added compliance phase tracking on the court dashboard
- Improved court violation health view layout

## Document Vault

- Improved document vault file handling and booking-related file management in the UI
