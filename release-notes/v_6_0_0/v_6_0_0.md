## General Changes

- Made parking violations no longer require a person record
- Fixed call sheet showing address as UNKNOWN when attached to an incident
- Updated module number to use the agency date and time from the server
- Made other small improvements and fixes

## Court Violations

- Introduced the Court Violation State Machine, with comprehensive procedural states (pre-plea, post-plea pre-judgment, FTA, FTA warrant, CPF warrant, bond hearing, and more)
- Added a financial event pipeline so payments and credits trigger state changes
- Added disposition plan operations and compliance tracking with due-date and extension logic
- Added the ability to vacate a judgment (requires an event note)
- Added the Plea Form to needs-review actions
- Added installment plans, the ability to add fees, jail-time credit, and balance logic for court violations
- Added Citation Front Desk Payment route
- Added the ability to print receipts for court violations
- Updated court documents (deferred disposition, installment agreement, show cause, and others)
- Added court violation search filters and report filtering, including by DPS code
- Added agency court settings (including IsUnder25)

## Online Payment & Stripe

- Stubbed in the Online Payment system with juvenile-aware logic
- Added Stripe webhook with hardened security
- Added accounting claims and roles
- Added accounting transaction-set search and reconciliation logic

## OCA Reporting

- Updated OCA XML generator, serializer, and report layout

## Mobile Citation

- Major sync logic updates with one-time clean-up of stuck citations and prevention of overwriting issued citations during sync
- Improved citation list reactivity
- Updated mobile citation styles, dynamic spacing, and agency preference overrides
- Updated mobile citation validations, including citation date/time and DOB
- Cleaned up parking violation validations and removed DL citation validation
- Updated driver license scan handling, including barcode parsing and version-number check
- Converted DL text to uppercase
- Removed the option to create or link a citation from a call sheet (with an option to include in regular citation numbering)

## Citation

- Updated citation logic and reports
- Added officer certification logic
- Updated complaint and offense mapping logic
- Updated next record-number stored procedure
- Added citation layout and validation updates

## Search

- Fixed racial profiling and impound log date-range searches
- Cleaned up search criteria logic
- Added master indices claims around search

## NIBRS & Offenses

- Merged multiple UCR offenses into one segment
- Fixed VO relationship validation
- Added fingerprint logic to offenses
- Added new offense codes and DPS code in search and results
- Added business day / calendar day selection for initial appearance dates
- Updated default child obligation notice

## Architecture Changes

- Upgraded the application to .NET 10
- Stubbed in .NET Aspire projects (with Vue) and converted the API to the minimal hosting model
- Migrated to the new `.slnx` solution file format
- Updated all NuGet packages and resolved warnings
- Configured Playwright for end-to-end testing
- Updated content security policy
- Switched to invariant string comparisons in several places
- Renamed async methods to follow C# conventions
- Added `AsNoTracking` to additional read-only queries
- Updated client-side libraries
- Updated server-side libraries
- Made other small improvements and fixes
