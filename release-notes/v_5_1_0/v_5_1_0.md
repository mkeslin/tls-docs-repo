## General Changes

- Updated auto master searches ("enter" still searches)
- Added analytics
- Updated rich text editor control
- Fixed issues with collapsible lists
- Updated user guide and release notes to use new Git repository
- Made other small improvements and fixes

## Architecture Changes

- Updated client-side libraries
- Updated server-side libraries
- Made other small improvements and fixes

# Version 5.1.1

- Added saving indicator on module workflow changes
- Added text search to unit/dispatch note search options

# Version 5.1.2

- Improved master merge functionality
- Improved loading indicator for module searches
- Updated CAD note edit dialog to support multi-line notes
- Updated Mobile Citation boilerplate text to match Texas statute
- Cleaned up speeding offense descriptions
- Updated/moved user feedback widget
- Updated and moved toast notifications
- Updated record information displayed for master records
- Updated master detail reports to match UI
- Added extended notes to CAD call sheets
- Added citations analytics and improved other analytics data
- Moved incident reports to analytics module

# Version 5.1.3
- Added additional loading indicators to search results
- Added additional saving/loading indicators to module add pages
- Added additional loading indicators to mobile citation import

# Version 5.1.4
- Fixed extended not display in incident details
- Added scrolling to CAD note edit dialogs

# Version 5.1.5
- Updated analytics logic to respect UTC times
- Fixed issue with selecting master records in CAD when entered text matches multiple options

# Version 5.1.6
- Fixed text color for read-only incident narratives
