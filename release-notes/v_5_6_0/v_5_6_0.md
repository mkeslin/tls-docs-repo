## General Changes

- Added new Court Violation module (initial implementation)
- Migrated court facilities to agencies and added court properties to agency administration
- Added court appearance date logic that uses the citation date instead of "now"
- Added probable cause logic and warning indicators on citations
- Stubbed in initial Accounting module
- Added agency type to support court agencies
- Added the ability to remove a citation that is stuck in sync
- Added the ability to print a citation when no person is on the record
- Added duplicate-master-association protection
- Refactored master organization and master location lookups
- Updated master cards and master search responsiveness
- Added DPS-specific fields to offenses
- Added additional Segment 9 IBRS validations for Texas
- Fixed value-of-zero issue in numeric input control
- Made other small improvements and fixes

## Mobile Citation

- Major updates to mobile citation logic, including court facility selection, signature, and PDF attachment
- Added loading spinner, issue floating-action button, responsive design, and improved empty-record styling
- Added date and time inputs to mobile citations
- Improved current location logic and vehicle creation from registration
- Improved sync (PDF attached as a separate request) and added a UI block while generating or syncing
- Fixed mobile citation court change bug
- Made other small improvements and fixes

## CAD

- Fixed unit state issue
- Removed cache loading from startup to improve responsiveness

## Architecture Changes

- Added new agency configurations
- Reorganized server projects, namespaces, and folders
- Updated Azure build, deploy production, and canary pipelines
- Added synopsis generation
- Updated client-side libraries
- Updated server-side libraries
- Made other small improvements and fixes
