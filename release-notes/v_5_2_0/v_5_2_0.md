## General Changes

- Added Adult/Juvenile Agency ID Numbers
- Moved master search logic to use new tables to increase performance
- Refactored IBRS tables to improve performance
- Moved audit timeline to new table to increase performance
- Added state code to offense search
- Added text highlighting to offense search
- Fixed search for admin offense table
- Added ability to add person/organization email/phone/alias with *enter* key
- Made other small improvements and fixes

## Architecture Changes

- Updated client-side libraries
- Updated server-side libraries
- Made other small improvements and fixes
