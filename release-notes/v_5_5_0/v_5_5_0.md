## General Changes

- Added the ability associate persons to other persons
- Added the ability to specify relationship types for master-to-master associations
- Added task list filter based on type
- Added new input controls for mobile citations
- Updated internal notes so they are editable even if record is closed/completed
- Updated incident report to print primary narrative first
- Updated "Add User" wizard to add Unit information
- Added dialog to make off-boarding users easier
- Split CAD permissions into full/self-dispatch roles/claims
- Updated cross-agency permissions for master records
- Made other small improvements and fixes

## Architecture Changes

- Updated client-side libraries
- Updated server-side libraries
- Made other small improvements and fixes
