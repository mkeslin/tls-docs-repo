# Vehicles

The shared **Vehicles** master for plates, VINs, and vehicle identity.

## Open and edit

1. Right rail → **Vehicles**.
2. Search by plate, VIN, make/model, and related fields.
3. Open the vehicle card. Common tabs:

| Tab | Contents |
|-----|----------|
| **General** | Category, year, make, model, style, colors, VIN, license plate # / state / country |
| **Records** | Module records that used this vehicle |
| **Associations** | Linked persons and organizations |
| **Notes** / **Attachments** / **History** | Same pattern as other masters |

## Tips

- Always include **plate + state** (and VIN when known) before creating a new vehicle.
- Citations and incidents link vehicle snapshots — fix the live master, then understand module copies may lag until updated — see [Linking and snapshots](linking-and-snapshots.md).
- Associate the registered owner **Person** or **Organization** when known.
