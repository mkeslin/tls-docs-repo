# Organizations

The shared **Organizations** master for businesses and other organizations.

## Open and edit

1. Right rail → **Organizations**.
2. Search by name / type (and advanced fields as shown).
3. Open the organization card. Common tabs:

| Tab | Contents |
|-----|----------|
| **General** | Name, organization type, and related identity fields |
| **Records** | Module records linked to this organization |
| **Associations** | Linked locations, vehicles, property, persons |
| **Notes** / **Attachments** / **History** | Same pattern as other masters |

## When to use an organization

Use **Organizations** when the party is a business, agency, or group — not when the party is an individual (**Persons**). Incident involved parties and similar screens often allow either a person or an organization depending on role.

## Tips

- Search before add — store names vary (“Joe’s Garage” vs “Joes Garage LLC”).
- Associate a **Location** when the org has a known address.
- Keep attachments for licenses or correspondence on the org master when policy allows.
