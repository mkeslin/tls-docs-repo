# Getting around

How to navigate **Jail Intake** and the **Jail Command Center**.

## Two places to work

| Where | How to open | Typical use |
|-------|-------------|-------------|
| **Jail Command Center** | Header product mode **JAIL** | Live floor ops: roster, moves, passes, release |
| **Jail Intake** | Left-rail **Jail Intake** (in RMS) | Booking Search, Booking Add, Pass History, Medication Log, Manage Locations |

Many agencies start a booking from the Command Center (**Add Booking**), finish steps in Intake, then return to the board after **Accept into Custody**.

From the Command Center, **RMS Records** jumps to Booking Search when you need history without leaving the jail context for long.

## Jail Intake menu (facility agencies)

| Destination | Purpose |
|-------------|---------|
| **Booking Search** | Find bookings by person, status, agency |
| **Booking Add** | Create a booking (wizard) |
| **Pass History** | Search completed observation / cell-check passes |
| **Medication Log** | Search medication administration history |
| **Manage Locations** | Housing and movement locations — [Manage locations](manage-locations.md) |

**Visitor Log**, Booking Health, and JLM Settings may be absent or support-only — do not expect them in every agency menu.

Partner / arresting-agency users may only see Booking Search and Booking Add (no Accept, no floor tools).

## Permissions

| Capability | Typical meaning |
|------------|-----------------|
| **Jail Intake — Access** | Command Center, search, locations, medication log |
| **Jail Intake — Modify** | Booking Add, complete steps, **Release**, related edits |
| **Jail Intake — Approve** | Add/edit episode medications (and related approve paths) |
| **Jail Observation — Access** | Pass History |
| **Jail Observation — Modify** | **Start Pass** / run Pass Runner |

**Accept into Custody** appears only when the app agency is a **jail facility** type — not a typical PD-only agency.

## Tips

- Keep the **same agency / facility** selected in the header when moving between Intake and **JAIL** mode.
- Use **Booking Search** for history; use the Command Center for “who is here right now.”
- If the board is empty, confirm agency selection before assuming there are no inmates.
