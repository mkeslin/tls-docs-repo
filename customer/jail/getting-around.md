# Getting around

How to navigate Jail Intake and the Jail Command Center.

## Two places to work

| Where | Typical use |
|-------|-------------|
| **Jail Command Center** | Live floor ops: who is in custody, moves, checks, release |
| **Jail Intake** | Booking search, add booking, complete intake steps, manage locations |

Many agencies start a booking from the Command Center (**Add Booking**), finish steps in Intake, then return to the board after **Accept Into Custody**.

## Main areas

| Area | What it is for |
|------|----------------|
| **Command Center** | Roster / layout board, In Booking Process, work queue, episode drawer |
| **Booking Search** | Find bookings by person, status, agency, and related filters |
| **Booking Add / Details** | Create a booking and walk the intake steps |
| **Pass History** | Search and open observation / cell-check pass records |
| **Medication Log** | Search medication administration history |
| **Manage Locations** | Configure housing and movement locations (jail-facility roles) |

Some support-only tools (for example booking health or advanced JLM settings) appear only for full-support or facility-admin users. Visitor log may be unavailable depending on your agency’s feature configuration.

## Permissions

What you can see and change depends on agency claims. Common ones include:

| Capability | Typical meaning |
|------------|-----------------|
| **Jail Intake — Access** | Open intake search and view bookings; open Command Center |
| **Jail Intake — Modify** | Create/edit bookings, complete steps, accept/release paths you are allowed |
| **Jail Observation — Access** | Work with observation passes and related history |

If a menu item or action is missing, ask your agency administrator to review jail roles.

## Tips

- Keep the **same agency / facility** selected when moving between Intake and the Command Center so lists and the board stay aligned.
- The Command Center updates in near real time when other staff move people or complete tasks — refresh only if something looks stuck after a network interruption.
- Use **Booking Search** when you need history; use the **Command Center** for “who is here right now.”
