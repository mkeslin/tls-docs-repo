# Command Center basics

![Jail Command Center](images/jail-command-center.png)

Day-to-day use of the **Jail Command Center** (header **JAIL**).

## Open the board

1. Switch the header to **JAIL**.
2. Select the correct **agency / facility** if prompted.
3. Confirm heartbeat counts look plausible before moving or releasing anyone.

## Main board areas

| Area | Purpose |
|------|---------|
| **Heartbeat** | Metrics: **Active**, **Onsite**, **Offsite**, **Unassigned**, **Suicide Watch**, **Due Now**, **Overdue** — click to filter |
| Top actions | **Search by name**, **Notification sounds**, **Add Booking**, roster **View PDF** / **Download PDF**, **RMS Records** |
| **Roster** / **Layout** | List vs location-oriented views; **Hide Empty Locations** on Layout |
| Sections | **In Booking Process**, **Onsite**, **Offsite**, **Unassigned** |
| Left panel | **Passes** (Start Pass) and **Medications** (due / upcoming) |
| Location rail | Destinations for moves + **Live Feed** at the bottom |
| Episode drawer | Selected person — flags, alerts, location, meds, Move / Release / Transfer out |

**Transfer Log** is not currently shown on the board in the shipping UI — use search/history paths your agency documents instead.

## Roster vs Layout

| View | Use when |
|------|----------|
| **Roster** | Scan who is in custody; open an episode quickly; **Sort: Name** / **Sort: Location** |
| **Layout** | See occupancy by cell/location for housing |

## In Booking Process

- See draft / in progress / ready bookings
- **Open booking** to finish steps or **Accept into Custody**
- Spot ready work waiting on intake

## Passes and Medications (left panel)

| Panel | Actions |
|-------|---------|
| **Passes** | **Start Pass** → choose **Pass type** (**CELL CHECK** / **SUICIDE CHECK**), **Group**, **Start** — see [Cell checks](observation-and-cell-checks.md) |
| **Medications** | **Next scheduled** / **Upcoming (next 90 minutes)** — see [Flags, alerts, and medication](flags-alerts-medication.md) |

## Filters

Heartbeat chips and banners such as **Filtering: Suicide Watch Only**. Use **Clear filters** when you need the full board.

## Live updates

Moves and tasks from other users refresh the board. If the connection drops, reconnect and re-confirm the roster before release or transfer.
