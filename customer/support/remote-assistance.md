# Remote assistance

When Thin Line needs to see your screen to diagnose an issue.

## How a session works

1. An **authorized Thin Line team member** asks you to start a remote session and gives you a session code.
2. On your computer, open **https://thinlinesoftware.screenconnect.com/**.
3. Enter the code they provide.
4. On first use, you may need to download and install the ConnectWise / ScreenConnect agent when prompted.

## Before you connect

- Close unrelated confidential documents and email.
- Stay with the computer during the session.
- Only accept remote sessions that Thin Line initiated through a known support channel.

## Security

If someone claiming to be Thin Line asks you to connect and you did **not** open a support case with Thin Line, stop and call the main office (**806-300-0455**) to verify.
