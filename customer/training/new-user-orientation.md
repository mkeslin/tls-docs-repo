# New user orientation

**Audience:** Any new Thin Line user  
**Goal:** Log in, recognize the shell, open modules, find help  
**Typical length:** ~2 hours  
**Environment:** Training tenant preferred  

## Prerequisites

- User account created; temporary password available
- Agency URL known
- Projector or shared screen for demo

## Agenda

| Block | Minutes | Guide | Exercise |
|-------|--------:|-------|----------|
| Welcome & access | 15 | [Logging in](../getting-started/logging-in.md) | First login + password change |
| Shell tour | 20 | [Application shell](../getting-started/application-shell.md) | Point to five zones; switch RMS / CAD / JAIL if licensed |
| Dashboard | 15 | [Dashboard](../getting-started/dashboard.md) | Open Tasks; find Report History; note What’s New |
| Modules | 20 | [Modules and navigation](../getting-started/modules-and-navigation.md) | Open two modules you will use; run a Search |
| Journey preview | 15 | [Journeys](../getting-started/journeys/README.md) | Walk the journey that matches their job (LE / Court / CAD / Jail) |
| Master records | 20 | [Master records](../getting-started/master-records/README.md) | Search a person; discuss “search before add” (full session: [Master records workshop](master-records-workshop.md)) |
| Header & agency | 15 | [Header and user menu](../getting-started/header-and-user-menu.md) · [Working across agencies](../getting-started/working-across-agencies.md) | Confirm agency; switch if multi-agency; open Help |
| Support path | 10 | [Support](../support/README.md) · [Contact](../support/contact.md) | List what belongs to agency admin vs Thin Line |
| Wrap-up | 5 | — | Questions; next workshop (Court / Jail / RMS) |

## Exit checklist

- [ ] User can log in and log out
- [ ] User can name the five shell zones
- [ ] User can open Dashboard Tasks and Report History
- [ ] User can open their primary module and run Search
- [ ] User can name which [journey](../getting-started/journeys/README.md) matches their job
- [ ] Multi-agency users can explain agency vs RMS/CAD/JAIL mode ([Working across agencies](../getting-started/working-across-agencies.md))
- [ ] User knows who resets passwords and how to reach Thin Line
