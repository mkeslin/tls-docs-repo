# CAD workshop

**Audience:** Dispatchers, call-takers, and officers who self-dispatch  
**Goal:** Run the live board, assign and clear units, dispose a call, and create an incident from CAD  
**Typical length:** ~3 hours  
**Environment:** Training tenant with CAD enabled and practice units

## Prerequisites

- [New user orientation](new-user-orientation.md)
- [Dashboard](../getting-started/dashboard.md) tour (Tasks + Self-Dispatch visibility)
- Accounts: at least one full-CAD dispatcher and one self-dispatch officer (unit-assigned)
- [Journey: CAD call to incident](../getting-started/journeys/cad-call-to-incident.md)

## Agenda

| Block | Minutes | Guide | Exercise |
|-------|--------:|-------|----------|
| Open console | 15 | [Live CAD overview](../cad/live-cad-overview.md) | Point to Units / Calls / Call Sheet |
| Create call | 25 | [Create and update a call](../cad/create-and-update-a-call.md) | Add Call; set type, priority, location |
| Units | 30 | [Assign and clear units](../cad/assign-and-clear-units.md) · [Unit status](../cad/unit-status.md) | Drag-assign; En Route/Arrived/Clear (as enabled) |
| Call sheet | 25 | [Call sheet activity](../cad/call-sheet-activity.md) | Add note + person + vehicle |
| Dispose | 15 | [Dispose and close a call](../cad/dispose-and-close-a-call.md) | Clear all units; set Disposition |
| Incident | 25 | [Related incidents and citations](../cad/related-incidents-and-citations.md) | Create Incident; open in RMS |
| Self-dispatch | 25 | [Self-dispatch](../cad/self-dispatch.md) | Officer assigns self; clears from call |
| Dispatcher notes & records | 15 | [Dispatcher notes](../cad/dispatcher-notes.md) · [CAD Records](../cad/records/README.md) | Add board note; find call in Call Sheets |
| Wrap | 5 | — | Questions |

## Exit checklist

- [ ] Open header **CAD** and name the three panels
- [ ] Add a call with Call Type and Location
- [ ] Assign and clear a unit (and En Route/Arrived if shown)
- [ ] Dispose a call with Disposition after all units cleared
- [ ] Create Incident from Related Incidents & Citations
- [ ] Officer completes a self-dispatch assign → clear
- [ ] Find the closed call in CAD Records Call Sheets
