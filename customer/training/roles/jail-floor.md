# Role plan: Jail floor / Command Center

**Audience:** Floor officers and supervisors on the Jail Command Center  
**Typical length:** Orientation + Jail workshop Session B (or combined half day)

## Goals

- Run Roster/Layout and filters
- House and move inmates
- Start/complete CELL CHECK and SUICIDE CHECK passes
- Log medication administration when assigned
- Release or complete offsite transfer

## Path

| Order | Curriculum / guide | Length |
|------:|--------------------|--------|
| 1 | [Shared foundation](README.md#shared-foundation-all-roles) | ~half day |
| 2 | [Command Center basics](../../jail/command-center.md) | 30 min |
| 3 | [Jail operations workshop](../jail-operations-workshop.md) — **Session B — Floor** | 2–3 h |
| 4 | [Manage locations](../../jail/manage-locations.md) — supervisors/admins | optional |

## Claims to verify

- Jail Intake Access (+ Modify for release/transfer)
- Jail Observation Modify for **Start Pass**
- Jail Approve if editing medication catalog

## Exit checklist

- [ ] Filter heartbeat metrics and clear filters
- [ ] Complete one housing **Move**
- [ ] Start and Complete a pass
- [ ] Log ADMINISTERED or UNABLE TO ADMINISTER (if role includes meds)
- [ ] Walk Release or Offsite Transfer in training
