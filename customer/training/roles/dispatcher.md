# Role plan: Dispatcher / call-taker

**Audience:** Full-console CAD dispatchers and call-takers  
**Typical length:** Orientation + ~half day CAD

## Goals

- Run the three-panel CAD console
- Create calls, assign/clear units, dispose with Disposition
- Create Incident from a call when reportable
- Use Dispatcher Notes and CAD Records appropriately

## Path

| Order | Curriculum / guide | Length |
|------:|--------------------|--------|
| 1 | [Shared foundation](README.md#shared-foundation-all-roles) | ~half day |
| 2 | Journey: [CAD call to incident](../../getting-started/journeys/cad-call-to-incident.md) | 30 min |
| 3 | [CAD workshop](../cad-workshop.md) | ~3 h |
| 4 | [Incidents](../../rms/incidents/README.md) — open created incidents | 30–60 min |

## Claims to verify

- Full CAD access + modify
- Related incidents / combined modify as needed
- Incidents access to open created reports

## Exit checklist

- [ ] Add Call with Call Type and Location
- [ ] Drag-assign and clear a unit (En Route/Arrived if enabled)
- [ ] Dispose a call
- [ ] Create Incident from Related Incidents & Citations
- [ ] Find the call later in CAD Records (not on the live board)
