# Role plan: Patrol officer

**Audience:** Patrol / field officers who cite, self-dispatch, and write reports  
**Typical length:** Orientation + ~1 day of workshops (can split across shifts)

## Goals

- Search before add on people/vehicles
- Issue citations (desktop or mobile)
- Self-dispatch and clear from a call when enabled
- Open or complete an incident when required

## Path

| Order | Curriculum / guide | Length |
|------:|--------------------|--------|
| 1 | [Shared foundation](README.md#shared-foundation-all-roles) | ~half day |
| 2 | Journey: [Law enforcement stop to report](../../getting-started/journeys/law-enforcement-stop-to-report.md) | 30 min |
| 3 | [Citations workshop](../citations-workshop.md) | 2–3 h |
| 4 | [CAD workshop](../cad-workshop.md) — Self-dispatch blocks | ~1 h (or full if also dispatching) |
| 5 | [Incidents workshop](../incidents-workshop.md) — light / agency policy | as needed |

## Claims to verify before class

- Citations modify (and mobile citations if used)
- CAD self-dispatch + unit assignment
- Incidents access/modify as policy requires
- Master search (and merge only if policy grants)

## Exit checklist

- [ ] Find and use an existing master person/vehicle
- [ ] Issue a training citation Draft → Issued
- [ ] Complete a self-dispatch assign → clear (if enabled)
- [ ] Know when to create an incident vs citation-only
- [ ] Know [Support](../../support/README.md) vs agency admin
