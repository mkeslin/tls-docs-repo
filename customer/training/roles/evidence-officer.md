# Role plan: Evidence / property room

**Audience:** Evidence custodians and officers who book property into evidence  
**Typical length:** Orientation + Evidence workshop (+ Incidents property segment)

## Goals

- Create evidence from incident property
- Maintain chain of custody
- Run inventory and inventory audit
- Understand master property vs evidence

## Path

| Order | Curriculum / guide | Length |
|------:|--------------------|--------|
| 1 | [Shared foundation](README.md#shared-foundation-all-roles) | ~half day |
| 2 | [Incidents — Property and evidence](../../rms/incidents/property-and-evidence.md) | 45 min |
| 3 | [Evidence workshop](../evidence-workshop.md) | ~2 h |
| 4 | Journey LE path — evidence segment | 20 min |

## Claims to verify

- Evidence access/modify as required
- Incidents access to create from property
- Inventory / audit claims if separated

## Exit checklist

- [ ] Add property on an incident and take into evidence
- [ ] Add a custody history event
- [ ] Search evidence and expand custody
- [ ] Run or explain Inventory Audit
