# Role plan: Records clerk

**Audience:** Records / RMS clerks who finish reports, warrants, and compliance exports  
**Typical length:** Orientation + 1–2 days

## Goals

- Own incident workflow quality and approvals handoff
- Maintain warrants and service attempts as assigned
- Run IBRS / racial profiling from Import/Export when responsible
- Keep masters clean (search before add; escalate merges)

## Path

| Order | Curriculum / guide | Length |
|------:|--------------------|--------|
| 1 | [Shared foundation](README.md#shared-foundation-all-roles) | ~half day |
| 2 | [Incidents workshop](../incidents-workshop.md) | ~3 h |
| 3 | [Warrants workshop](../warrants-workshop.md) | ~2 h |
| 4 | [Citations workshop](../citations-workshop.md) — if records finishes tickets | as needed |
| 5 | [Import/Export](../../import-export/README.md) + [data quality checklist](../../import-export/data-quality-checklist.md) | 1–2 h |
| 6 | [Evidence workshop](../evidence-workshop.md) — if records supports property room | optional |

## Claims to verify

- Incidents access/modify/approve as role requires
- Warrants access/modify
- IBRS and/or Racial Profiling report claims
- Master merge only if policy allows

## Exit checklist

- [ ] Complete or advance a training incident through approval path
- [ ] Add a service attempt on a warrant
- [ ] Create/validate an IBRS or RP run in training (no external submit)
- [ ] Demonstrate search-before-add
