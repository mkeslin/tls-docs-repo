# Grant a court program

![Court program on a case](../images/court-programs.png)

## Goal

Grant a deferred disposition / diversion-style **court program** with conditions and due dates.

## Prerequisites

- Case on a path that allows granting a program (often after a guilty / no contest style plea, before final judgment)
- Court order authorizing the program
- Modify rights on Court Violations

## Steps

1. Open the court violation.
2. Confirm the plea path is complete (enter plea first if needed — [Record a plea and judgment](record-a-plea-and-judgment.md)).
3. Choose **Grant court program** (or **Grant deferred disposition** — use the label in your build).
4. Enter program type, conditions, attorney information if used, and due dates.
5. Save / confirm.
6. Use **View/modify court program** later when you only need to edit conditions without changing state.

## Expected result

- Case is on the court program / compliance track.
- Condition due dates drive program work queues (missed deadline, ready to close, show cause).
