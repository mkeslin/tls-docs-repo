# Opening Import/Export

1. Open **Import/Export** from the left module rail.
2. Choose the export you are authorized to run.
3. If an item is missing, you lack the claim or module access — ask your administrator ([Admin — Users](../admin/users-roles-claims.md)).

## Who sees what (high level)

| Menu item | Needs |
|-----------|--------|
| **IBRS** | IBRS / NIBRS report access (+ incidents context) |
| **Racial Profiling** | Racial profiling report access |
| **DPS Conviction / OCA / State Quarterly** | Court access + court violation report claim |

You may see only a subset of the menu.
