# OCA Report

![Import/Export reports](images/import-export-ibrs.png)

Monthly Office of Court Administration–style reporting pack.

## Prerequisites

Agency must have an **OCA court identifier** configured. If create is blocked for missing identifier, escalate to Admin / Thin Line.

## Create and download

1. Open **Import/Export** → **OCA Report**.
2. **Create** for **Year** and **Month**.
3. Open the report → **Rebuild** if needed → **Download** (PDF may be available depending on build flags).
4. Retain history for audit.
