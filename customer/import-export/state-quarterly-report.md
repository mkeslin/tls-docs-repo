# State Quarterly Report

![Import/Export reports](images/import-export-ibrs.png)

Quarterly court statistical pack.

## Create and download

1. Open **Import/Export** → **State Quarterly Report**.
2. **Create** — agency, **Report Year**, **Report Quarter** (`Q1`–`Q4`).
3. Open the report → **Rebuild** if needed → **Download** (summary PDF when offered).
4. Submit per your state’s quarterly calendar.
