# CAD

Customer guides for Thin Line **CAD** — the live dispatch console — plus **self-dispatch** and **CAD Records** for historical search.

## What CAD is

**CAD** (computer-aided dispatch) is the real-time console for calls for service and unit status. Open it from the header **CAD** product mode (not from the left-rail module list).

| Destination | Purpose |
|-------------|---------|
| Header **CAD** | Live three-panel console — Units, Calls, Call Sheet |
| [Dashboard](../getting-started/dashboard.md) → **Self-Dispatch** | Officers assign themselves and update their own unit |
| Left-rail **CAD Records** | Historical call sheets, unit logs, dispatcher notes (not live dispatch) |

## Who this guide is for

- Dispatchers and call-takers on the full console
- Officers using self-dispatch
- Records staff searching historical CAD data
- Trainers running CAD go-live sessions
