# CAD

Customer guides for Thin Line **CAD** — the live dispatch console — and **CAD Records** for historical search inside RMS.

## What CAD is

**CAD** (computer-aided dispatch) is the real-time console for calls for service and unit status. Open it from the header **CAD** product mode (not from the left-rail module list).

| Destination | Purpose |
|-------------|---------|
| Header **CAD** | Live console — units and active calls |
| Left-rail **CAD Records** | Historical call sheets, unit logs, dispatcher notes, and related searches |
| Dashboard self-dispatch (when enabled) | Officers create/self-assign work without the full dispatch console |

## Who this guide is for

- Dispatchers and call-takers
- Officers using self-dispatch
- Records staff searching historical CAD data
