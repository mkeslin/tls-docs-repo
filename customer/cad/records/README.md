# CAD Records

Historical CAD search inside the RMS shell (left-rail **CAD Records**).

## What CAD Records is

CAD Records is **not** the live dispatch console. Use it to research past call sheets, unit activity, dispatcher notes, and related logs without taking over the CAD board.

Requires CAD Records access (LETS Queries also needs LETS query access).
