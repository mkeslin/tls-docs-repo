# Add evidence wizard

Creating evidence from an incident using the property / evidence stepper.

## Where it starts

From an open **Incident**:

- **Evidence** tab → **Add Evidence**, or  
- **Property** tab → **Add as Evidence** / related wizard entry points  

Exact button labels follow your build; the wizard typically offers modes such as:

| Mode (concept) | What it does |
|----------------|--------------|
| Add property only | Property on the case, not yet evidence |
| Add property and take into custody as evidence | Creates property + first **IN AGENCY CONTROL** history |
| Take existing property into custody as evidence | Promotes an existing property line into evidence |

## After the wizard

1. Confirm the new **Evidence #** appears on the incident Evidence tab.
2. Verify the first custody history (officer vs property room, location, notes).
3. Print a **Label** or **Chain of Custody** if your room requires it immediately — see [Print and labels](print-and-labels.md).

## Tips

- You need **modify evidence** rights (and incident modify as required) to complete intake.
- Do not create a second evidence number for the same physical item to “fix” a bad first entry — add a correcting history or ask a supervisor with admin evidence rights.
- Package and seal physical evidence per agency SOP; the software records custody, it does not replace packaging standards.
