# Print and labels

Evidence prints from the Evidence module and from the incident.

## From Search / Inventory

- Grid / list **PDF** for the current filter or inventory view
- From an item’s history actions (when shown): **Chain Of Custody**, **Label**

## From the incident

On **Incident → Evidence**, print controls commonly include:

| Print | Use |
|-------|-----|
| **In Agency Control Form** | Document items in custody for the case |
| **Out of Agency Control Form** | Document release / out items (selector when multiple) |
| **All Evidence For Incident** | Full case evidence list |
| Per-item **Chain Of Custody** / **Label** | Same as module history prints when offered |

## Inventory audit

See [Inventory audit](inventory-audit.md) for the random-sample audit report.

## Tips

- Print labels when the item is sealed and placed — not days later from memory.
- Keep CoC prints with the case file per policy.
- Re-print after correcting history so paper matches the system.
