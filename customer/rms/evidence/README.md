# Evidence

![Evidence detail](images/rms-evidence-detail.png)

Customer guides for the **Evidence (EVI)** module and incident-based evidence custody in Thin Line RMS.

## What Evidence is

Evidence in Thin Line is **property taken into agency control** with a numbered **evidence** record and a **chain of custody** (history of in / out / transfer events). Most create and custody work happens from an **Incident**, not from a standalone Evidence “detail” page.

| Place | What you do there |
|-------|-------------------|
| **Incident → Property / Evidence** | Add property, take into custody as evidence, add custody history, print case evidence forms |
| **Evidence module** | **Search**, **Inventory**, and **Inventory Audit** across the property room |
| **Master property** | See evidence numbers linked to a property master |

Nav abbreviation is **EVI** (module code may show as EVC in technical contexts). For customers, use **Evidence**.

## Who this guide is for

- Evidence / property room staff
- Officers who book property into evidence from a case
- Supervisors who run inventory and audits
- Records staff who search evidence by number or incident
