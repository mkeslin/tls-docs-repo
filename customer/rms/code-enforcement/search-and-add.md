# Search and add code enforcement cases

## Search

1. Open **Code Enforcement** from the left rail.
2. Choose **Search**.
3. Enter case number, location, person, status, or dates as shown.
4. Open a result to work the case.

## Add

1. Choose **Add** (requires modify rights).
2. Complete required intake fields (location and complaint context).
3. Link [master locations and persons](../../getting-started/master-records/README.md) — search before add.
4. Save, then continue on [Working a case](working-a-case.md).

## Tips

- One case per problem location / complaint stream avoids duplicate timelines.
- Delete (when your role allows) is destructive — prefer status changes for closed work unless your agency policy says otherwise.
