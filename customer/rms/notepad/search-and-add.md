# Search and add notepads

![Notepad search](images/rms-notepad-search.png)

## Search

1. Open **Notepad** from the left rail.
2. Choose **Search**.
3. Enter criteria and run the search.
4. Open a result to work the notepad.

## Add

1. Choose **Add** (requires notepad modify rights).
2. Complete required general fields.
3. Link masters (person, vehicle, location, property) with [search before add](../../getting-started/master-records/search-and-add.md).
4. Save, then complete [Working a notepad](working-a-notepad.md) tabs.

## Tips

- Search first — duplicate notepads for the same contact confuse follow-up.
- Lost & Found has its own search entry under the Notepad menu — see [Lost and found](lost-and-found.md).
