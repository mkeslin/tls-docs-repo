# RMS

Customer guides for Thin Line **Records Management** modules — day-to-day records work in the left module rail.

## Module guides

| Module | Guide |
|--------|-------|
| [Citations](citations/README.md) | Tickets, mobile citations, citation → court |
| [Incidents](incidents/README.md) | Incident reports, arrests, evidence from property |
| [Warrants](warrants/README.md) | Local and court-owned warrants |
| [Evidence](evidence/README.md) | Search, inventory, custody |
| [Close Patrol](close-patrol/README.md) | Property checks / vacation watches |
| [Notepad](notepad/README.md) | Field contacts; Lost & Found |
| [Code Enforcement](code-enforcement/README.md) | Ordinance / code cases |

Fire Incidents may be enabled for some agencies; a dedicated customer guide is not published yet. Criminal Trespass, Equipment, and Fleet are intentionally omitted until product-ready.
